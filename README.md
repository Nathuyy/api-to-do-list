# to-do-list-fullstack
Este é um projeto completo de uma aplicação CRUD (Create, Read, Update, Delete) para gerenciar uma lista de tarefas. A aplicação é construída utilizando Node.js, Express e MySQL.

## Funcionalidades:
Adicionar uma nova tarefa
Listar todas as tarefas
Obter detalhes de uma tarefa específica
Atualizar uma tarefa
Excluir uma tarefa
